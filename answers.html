Brute Force Solutions to Array Questions

1. Find Two Numbers That Sum to \( x \)

Problem: Given an integer \( x \) and an array of numbers, find if two numbers in the array sum up to \( x \). (Or, find the pair whose sum is closest to \( x \).)

Brute Force Solution:

FUNCTION findPairSum(arr, x):
    n = length(arr)
    FOR i = 0 to n - 1:
        FOR j = i + 1 to n - 1:
            IF arr[i] + arr[j] = x:
                RETURN (arr[i], arr[j])
    RETURN NULL

Explanation: Check all pairs of elements to find two whose sum equals \( x \).

Runtime Analysis: O(n²) time, O(1) space.

2. Median of Union of Two Sorted Arrays

Problem: Given two sorted arrays, find the median of the union of elements.

Brute Force Solution:

FUNCTION findMedian(A, B):
    C = MERGE(A, B)
    n = length(C)
    IF n MOD 2 = 0:
        RETURN (C[n/2 - 1] + C[n/2]) / 2
    RETURN C[n/2]

FUNCTION MERGE(A, B):
    result = EMPTY array
    i = 0, j = 0
    WHILE i < length(A) AND j < length(B):
        IF A[i] <= B[j]:
            ADD A[i] to result
            i = i + 1
        ELSE:
            ADD B[j] to result
            j = j + 1
    WHILE i < length(A):
        ADD A[i] to result
        i = i + 1
    WHILE j < length(B):
        ADD B[j] to result
        j = j + 1
    RETURN result

Explanation: Merge the arrays into a sorted array and find the median based on the total length.

Runtime Analysis: O(n + m) time, O(n + m) space.

3. k-th Smallest in Union of Two Sorted Arrays

Problem: Given two sorted arrays, find the \( k \)-th smallest element in the union of elements.

Brute Force Solution:

FUNCTION findKth(A, B, k):
    C = MERGE(A, B)
    IF k > length(C) OR k < 1:
        RETURN NULL
    RETURN C[k - 1]

FUNCTION MERGE(A, B):
    result = EMPTY array
    i = 0, j = 0
    WHILE i < length(A) AND j < length(B):
        IF A[i] <= B[j]:
            ADD A[i] to result
            i = i + 1
        ELSE:
            ADD B[j] to result
            j = j + 1
    WHILE i < length(A):
        ADD A[i] to result
        i = i + 1
    WHILE j < length(B):
        ADD B[j] to result
        j = j + 1
    RETURN result

Explanation: Merge the arrays into a sorted array and return the \( k \)-th element.

Runtime Analysis: O(n + m) time, O(n + m) space.

4. k-th Smallest in Union of \( \ell \) Sorted Arrays

Problem: Find the \( k \)-th smallest element in the union of \( \ell \) sorted arrays. What is the running time?

Brute Force Solution:

FUNCTION findKthMulti(arrays, k):
    result = EMPTY array
    FOR each array in arrays:
        FOR each element in array:
            ADD element to result
    n = length(result)
    FOR i = 0 to n - 1:
        FOR j = i + 1 to n - 1:
            IF result[j] < result[i]:
                SWAP result[i] and result[j]
    IF k > n OR k < 1:
        RETURN NULL
    RETURN result[k - 1]

Explanation: Concatenate all arrays and sort them, then return the \( k \)-th element.

Runtime Analysis: O(n * log n) time for sorting (where \( n \) is total elements), O(n) space.

5. Merge \( k \) Sorted Arrays

Problem: Merge \( k \) sorted arrays into a single sorted array.

Brute Force Solution:

FUNCTION mergeKArrays(arrays):
    result = EMPTY array
    FOR each array in arrays:
        FOR each element in array:
            ADD element to result
    n = length(result)
    FOR i = 0 to n - 1:
        FOR j = i + 1 to n - 1:
            IF result[j] < result[i]:
                SWAP result[i] and result[j]
    RETURN result

Explanation: Concatenate all arrays and sort using bubble sort.

Runtime Analysis: O(n²) time (where \( n \) is total elements), O(n) space.

6. Single Element Appearing Once

Problem: Given an unsorted array where every integer appears twice except one, find it.

Brute Force Solution:

FUNCTION findSingle(arr):
    n = length(arr)
    FOR i = 0 to n - 1:
        count = 0
        FOR j = 0 to n - 1:
            IF arr[j] = arr[i]:
                count = count + 1
        IF count = 1:
            RETURN arr[i]
    RETURN NULL

Explanation: Count occurrences of each element; the one with count 1 is the answer.

Runtime Analysis: O(n²) time, O(1) space.

8. Nearest Index \( j > A[i] \)

Problem: For each \( i \), find the nearest \( j > i \) where \( A[j] > A[i] \), report -1 if none.

Brute Force Solution:

FUNCTION nearestLarger(arr):
    n = length(arr)
    result = ARRAY of length(n), fill with -1
    FOR i = 0 to n - 1:
        FOR j = i + 1 to n - 1:
            IF arr[j] > arr[i]:
                result[i] = j
                BREAK
    RETURN result

Explanation: For each \( i \), scan all \( j > i \) to find the first larger element.

Runtime Analysis: O(n²) time, O(n) space.

9. Move Zeros to Right

Problem: Move all zeros to the right end with \( O(1) \) extra storage.

Brute Force Solution:

FUNCTION moveZeros(arr):
    n = length(arr)
    FOR i = 0 to n - 1:
        FOR j = i + 1 to n - 1:
            IF arr[i] = 0 AND arr[j] ≠ 0:
                SWAP arr[i] and arr[j]
    RETURN arr

Explanation: Use bubble sort-like swaps to push zeros to the end.

Runtime Analysis: O(n²) time, O(1) space.

11. Median of Stream

Problem: Return the median of numbers seen so far in a stream.

Brute Force Solution:

FUNCTION getMedian(stream):
    numbers = EMPTY array
    WHILE stream has next:
        ADD next number to numbers
        n = length(numbers)
        FOR i = 0 to n - 1:
            FOR j = i + 1 to n - 1:
                IF numbers[j] < numbers[i]:
                    SWAP numbers[i] and numbers[j]
        IF n MOD 2 = 0:
            RETURN (numbers[n/2 - 1] + numbers[n/2]) / 2
        RETURN numbers[n/2]

Explanation: Store all numbers, sort after each addition, and compute the median.

Runtime Analysis: O(n²) per update, O(n) space.

13. Missing Element in 1 to n

Problem: Find the missing element from 1 to \( n \), or two missing if applicable.

Brute Force Solution:

FUNCTION findMissing(arr):
    n = length(arr) + 1
    missing = EMPTY list
    FOR i = 1 to n:
        found = false
        FOR j = 0 to length(arr) - 1:
            IF arr[j] = i:
                found = true
                BREAK
        IF NOT found:
            ADD i to missing
        IF length(missing) = 2:
            BREAK
    RETURN missing

Explanation: Check each number from 1 to \( n \) for presence, stop after two misses.

Runtime Analysis: O(n²) time, O(1) space.

15. Merge Intersecting Ranges

Problem: Merge intersecting ranges into disjoint ranges.

Brute Force Solution:

FUNCTION mergeRanges(ranges):
    changed = true
    WHILE changed:
        changed = false
        n = length(ranges)
        FOR i = 0 to n - 1:
            FOR j = i + 1 to n - 1:
                IF ranges[i][1] >= ranges[j][0]:
                    ranges[i][1] = MAX(ranges[i][1], ranges[j][1])
                    REMOVE ranges[j]
                    changed = true
                    BREAK
    RETURN ranges

Explanation: Repeatedly check and merge overlapping ranges until no overlaps remain.

Runtime Analysis: O(n²) per pass, O(n²) total in worst case, O(n) space.

16. Common Elements in Three Arrays

Problem: Find elements common to three sorted arrays.

Brute Force Solution:

FUNCTION findCommon(A, B, C):
    common = EMPTY list
    FOR i = 0 to length(A) - 1:
        FOR j = 0 to length(B) - 1:
            IF A[i] = B[j]:
                FOR k = 0 to length(C) - 1:
                    IF B[j] = C[k]:
                        ADD A[i] to common
    RETURN common

Explanation: Check each element of A against B, then C, for common elements.

Runtime Analysis: O(n³) time, O(k) space (k = number of common elements).

17. Extra Element in Two Arrays

Problem: Find the extra element and its index in two nearly identical sorted arrays.

Brute Force Solution:

FUNCTION findExtra(A, B):
    n = length(A)
    m = length(B)
    IF n = m:
        RETURN NULL
    FOR i = 0 to n - 1:
        found = false
        FOR j = 0 to m - 1:
            IF A[i] = B[j]:
                found = true
                BREAK
        IF NOT found:
            RETURN (A[i], i)
    RETURN NULL

Explanation: Check each element of A in B; the first not found is the extra element.

Runtime Analysis: O(n * m) time, O(1) space.

18. Closest Element in Sorted Array

Problem: Find the element closest to a given number in a sorted array.

Brute Force Solution:

FUNCTION findClosest(arr, target):
    n = length(arr)
    min_diff = ABS(arr[0] - target)
    closest = arr[0]
    FOR i = 1 to n - 1:
        diff = ABS(arr[i] - target)
        IF diff < min_diff:
            min_diff = diff
            closest = arr[i]
    RETURN closest

Explanation: Compare the absolute difference of each element with the target to find the closest.

Runtime Analysis: O(n) time, O(1) space.

19. Local Maximum in 1D Array

Problem: Find a local maximum \( A[i] \geq \) neighbors.

Brute Force Solution:

FUNCTION findLocalMax(arr):
    n = length(arr)
    FOR i = 0 to n - 1:
        is_max = true
        IF i > 0 AND arr[i] < arr[i - 1]:
            is_max = false
        IF i < n - 1 AND arr[i] < arr[i + 1]:
            is_max = false
        IF is_max:
            RETURN arr[i]
    RETURN NULL

Explanation: Check each element against its neighbors to find a local maximum.

Runtime Analysis: O(n) time, O(1) space.

20. Local Maximum in 2D Array

Problem: Find a local maximum \( A[i][j] \geq \) vertical and horizontal neighbors.

Brute Force Solution:

FUNCTION findLocalMax2D(matrix):
    n = length(matrix)
    FOR i = 0 to n - 1:
        FOR j = 0 to n - 1:
            is_max = true
            IF i > 0 AND matrix[i][j] < matrix[i-1][j]:
                is_max = false
            IF i < n - 1 AND matrix[i][j] < matrix[i+1][j]:
                is_max = false
            IF j > 0 AND matrix[i][j] < matrix[i][j-1]:
                is_max = false
            IF j < n - 1 AND matrix[i][j] < matrix[i][j+1]:
                is_max = false
            IF is_max:
                RETURN matrix[i][j]
    RETURN NULL

Explanation: Check each element against its four neighbors to find a local maximum.

Runtime Analysis: O(n²) time, O(1) space.

23. Maximize Stock Profit

Problem: Find maximum profit by buying and selling on subsequent days.

Brute Force Solution:

FUNCTION maxProfit(prices):
    n = length(prices)
    max_profit = 0
    FOR i = 0 to n - 1:
        FOR j = i + 1 to n - 1:
            profit = prices[j] - prices[i]
            IF profit > max_profit:
                max_profit = profit
    RETURN max_profit

Explanation: Check all buy-sell pairs to find the maximum profit.

Runtime Analysis: O(n²) time, O(1) space.

24. Best Selling Day for Each Day

Problem: For each day \( i \), find \( j > i \) maximizing \( A[j] - A[i] \).

Brute Force Solution:

FUNCTION bestSellDays(prices):
    n = length(prices)
    result = ARRAY of length(n)
    FOR i = 0 to n - 1:
        max_profit = 0
        best_j = i
        FOR j = i + 1 to n - 1:
            profit = prices[j] - prices[i]
            IF profit > max_profit:
                max_profit = profit
                best_j = j
        result[i] = best_j
    RETURN result

Explanation: For each day, scan future days to find the best selling day.

Runtime Analysis: O(n²) time, O(n) space.

27. Index \( i \) Where \( A[i] = i \)

Problem: Find an index \( i \) where \( A[i] = i \) in a sorted array.

Brute Force Solution:

FUNCTION findFixedPoint(arr):
    n = length(arr)
    FOR i = 0 to n - 1:
        IF arr[i] = i:
            RETURN i
    RETURN -1

Explanation: Check each index to see if \( A[i] = i \).

Runtime Analysis: O(n) time, O(1) space.

28. Number of Non-Zero Elements

Problem: Find \( n \) where the first \( n \) entries are positive and the rest are 0.

Brute Force Solution:

FUNCTION findNonZeroCount(arr):
    n = length(arr)
    FOR i = 0 to n - 1:
        IF arr[i] = 0:
            RETURN i
    RETURN n

Explanation: Scan until the first zero to determine \( n \).

Runtime Analysis: O(n) time, O(1) space.
